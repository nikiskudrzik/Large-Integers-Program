# Large Integers Program

## Description
This program uses a class called largeIntegers to store and work with very large numbers that have many digits. Since normal data types cannot store very large numbers, the program stores them as strings and performs math operations manually.

The program can:
- store large integers
- add two numbers
- subtract two numbers
- multiply two numbers
- compare two numbers

## Inputs
The program uses the following numbers:
- 123456789987654321
- 987654321123456789

## Outputs
The program prints:
- the first number
- the second number
- the sum of the numbers
- the difference of the numbers
- the product of the numbers
- the comparison result

## How to Compile and Run

Compile:
g++ main.cpp largeIntegers.cpp -o largeIntegersProgram

Run (Mac):
./largeIntegersProgram
